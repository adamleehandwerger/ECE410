## Corrected MAC Module Implementation

The corrected version of the MAC module:

```systemverilog
module mac (
    input  logic        clk,
    input  logic        rst,
    input  logic signed [7:0]  a,
    input  logic signed [7:0]  b,
    output logic signed [31:0] out
);

    always_ff @(posedge clk) begin
        if (rst)
            out <= 32'sd0;
        else
            out <= out + 32'(a * b);
    end

endmodule
```

### Key Features:
- **Signed arithmetic**: Uses `signed` for inputs `a` and `b` (8-bit) and output `out` (32-bit)
- **Synchronous reset**: Reset is synchronous to the clock edge
- **Accumulation**: Each cycle adds the product `a × b` to the current accumulator value
- **Width extension**: Properly extends 8×8=16-bit product to 32-bit before accumulation
